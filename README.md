# Capage
Image Capture

### Installation
```
  brew install tesseract
```

For other distros follow https://github.com/tesseract-ocr/tesseract/wiki

```
  pip install requests, flask, pytesseract
```

### Running

```
  python flask_server/app.py
```

App runs on localhost:5001
